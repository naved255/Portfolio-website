- 📧 EmailJS (for contact form)
- 🚀 Vercel (deployment)

- 🔥 Responsive and modern UI
- 🌙 Clean and minimal design
- 📬 Contact form with email functionality
- 🔗 Social links (GitHub, LinkedIn)
